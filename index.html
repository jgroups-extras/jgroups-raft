Welcome to jgroups-raft

The jgroups-raft project is a Raft implementation in JGroups. It is currently in its own repo because it'll move much faster than JGroups and I wanted to have the freedom to release versions in quick succession.

Documentation

Manual is available here.

Note that the manual can be generated locally. To do this, check out jgroups-raft and execute ant manual. The manual will be generated in the build directory. On my todo list: to update the web page automatically with the generated manual...

Use with maven

    <groupId>org.jgroups</groupId>
    <artifactId>jgroups-raft</artifactId>
    <version>1.1.3.Final</version>
