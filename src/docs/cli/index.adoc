Welcome to the official documentation for the **JGroups Raft CLI**.

This tool serves as the primary interface for operators and administrators to interact with JGroups Raft clusters.
Unlike standard network diagnostic tools, this CLI is built with deep knowledge of the Raft consensus protocol, enabling precise monitoring of terms, log replication, and leadership states.

This manual is organized into the following sections:

* **Overview**: Covers installation options, network configuration, and the fundamental concepts of utilizing the CLI.
* **Monitoring Commands**: Details the monitoring operations used to observe cluster health, leadership stability, and performance metrics.
* **Management Commands**: Explains administrative actions for modifying cluster membership and executing advanced probe requests for low-level diagnostics.
